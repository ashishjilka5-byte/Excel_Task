# 📊 Sales Performance & Predictive Analytics Dashboard

---

## 📌 Project Overview

Yeh project ek raw transactional dataset (250+ records) ko ek interactive **Business Intelligence dashboard** mein convert karne ke liye design kiya gaya hai.

Iska main objective:

* Sales trends ko track karna
* Top-performing regions ko identify karna
* Regression analysis ke through future insights nikalna

---

## 🚀 Key Performance Indicators (KPIs)

| Metric                   | Value       | Top Performer          |
| ------------------------ | ----------- | ---------------------- |
| 💰 Total Revenue         | $229,192.47 | 📍 East Region ($59k+) |
| 📦 Total Units Sold      | 753 Units   | Bookshelf (102 Qty)    |
| 👤 Top Customer          | Mark Carter | $15,659.65 Spend       |
| 💻 Highest Value Product | Laptop      | $67,499.25 Revenue     |

---

## ⚙️ Tools & Technologies Used

* Microsoft Excel (Data Cleaning, Pivot Tables, Dashboard)
* Statistical Analysis (Linear Regression - ANOVA)
* Data Visualization (Pivot Charts, Slicers, Timelines)
* Excel Formulas:

  * VLOOKUP
  * SUMIFS
  * DATEDIF
  * NOW
  * Advanced Date Functions

---

## 📊 Features of Dashboard

* 📈 Dynamic Charts → Region-wise & Category-wise analysis
* 📅 Time-based Analysis → Monthly & Quarterly trends
* 🔍 Interactive Slicers → Region, Category, Customer filtering
* 📉 Predictive Insights → Regression-based analysis

---

## 🔍 Deep-Dive Insights & Analysis

### 1️⃣ Market & Product Strategy

* East Region market leader hai ($59,288)
* South region ek potential growth market hai
* Bookshelves & Smartphones → High volume drivers
* Laptops → Highest revenue contributor

💡 **Inventory Insight:**

* Keyboard & Monitors ki demand stable hai
* Inka stock optimized rehna chahiye

---

### 2️⃣ Sales Momentum

* 2024 Total Sales → $160K
* 2025 Q1 → $61K

📈 Strong positive growth trend observed

---

## 🧠 Statistical Analysis (Regression Results)

* **R-Square = 0.21**
  → Model 21% sales variation explain karta hai

* **Significance F = 2.25 × 10⁻¹⁴**
  → Model statistically valid hai (not by chance)

---

## 🧮 Key Components

* Data Cleaning → Missing values & duplicates handle kiye
* Pivot Table Analysis → Structured data insights
* UI/UX Design → Clean & professional dashboard layout
